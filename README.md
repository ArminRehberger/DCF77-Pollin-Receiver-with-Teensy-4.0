# DCF77WithTeensy4.0AndPollinReceiver
DCF77 Receiver with Teensy4.0 and Pollin receiver

Pollin receiver

Pin 1 = VCC -> 3.3V

Pin 2 = GND

Pin 3 = TCO -> Data output

Pin 4 = PON -> connected to GND
