# DCF77 Pollin Receiver With Teensy4.0
DCF77 Pollin Receiver With Teensy4.0

Pollin receiver

Pin 1 = VCC -> 3.3V

Pin 2 = GND

Pin 3 = TCO -> Data output

Pin 4 = PON -> connected to GND
